= Recepten

Voor het publisheren van beproefde recepten.

== Recepten bewerken

In `_data/recepten` is per recept een yaml file aanwezig.

== De recepten site lokaal bekijken

Met behulp van de `Makefile` kun je de Jekyll site laten genereren.
Indien de git repo vers is opgehaald, draai dan eerst `make install`.
Gebruik daarna `make serve` om de site te maken, en lokaal beschikbaar te stellen.
Zie de `Makefile` voor de andere commando's.
